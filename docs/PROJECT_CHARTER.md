# رفيق العلوم — ميثاق المشروع

## الرؤية

رفيق العلوم منظومة تعلم تفاعلية موجهة للطالب، وليست بنك ملخصات أو صفحة أسئلة فقط.

يبني كل درس وفق التسلسل:

هدف التعلم → شرح مبسط → مثال → نشاط → سؤال → تغذية راجعة → قياس إتقان

## الفئة المستهدفة

طلبة مواد العلوم والفيزياء والكيمياء والأحياء.

نطاق MVP الحالي:

- الصف: العاشر
- المادة: الفيزياء
- الوحدة: الموجات
- عدد الدروس: 4 دروس فقط

## الاسم المرحلي

رفيق العلوم

## القواعد التأسيسية

1. لا كود قبل وضوح البنية.
2. لا ملف ضخم يجمع كل شيء.
3. فصل واجهة الطالب عن واجهة المعلم.
4. فصل المحتوى المعتمد عن المحتوى المولّد.
5. التطوير يتم على مراحل صغيرة قابلة للفحص.
6. لا Supabase قبل نجاح Prototype الطالب المحلي.
7. لا AI حي قبل مرحلة التأليف المعتمدة.
8. لا ألعاب تعليمية بلا هدف تعلم واضح.
9. لا تجارب عملية بلا safetyLevel.
10. RTL والتصميم جزء من الأساس، لا مرحلة تجميل لاحقة.
