# رفيق العلوم — قواعد المحتوى

## مستوى اللغة

اللغة مناسبة لطالب الصف العاشر، مباشرة، واضحة، ولا تستخدم مصطلحات جامعية غير ضرورية.

## صياغة الأهداف

الهدف يجب أن يبدأ بفعل قابل للقياس، مثل:

- يميّز
- يفسّر
- يطبّق
- يقارن
- يستنتج

ولا يفضّل استخدام صياغات عامة مثل: يفهم أو يعرف فقط.

## أسئلة الاختيار من متعدد

كل سؤال يجب أن يحتوي على:

- إجابة صحيحة واحدة فقط.
- مشتتات معقولة.
- ربط إلزامي بـ `objectiveId`.
- تغذية راجعة تشرح السبب.

## التغذية الراجعة

يجب أن تكون:

- فورية.
- غير حكمية على الطالب.
- قصيرة وواضحة.
- تشرح سبب صحة أو خطأ الإجابة.

## اللعبة التعليمية

لا تقبل أي لعبة إلا إذا:

- ارتبطت بهدف أو أكثر من `objectiveIds`.
- اختبرت مفهومًا حقيقيًا.
- لم تكن زخرفة أو ترفيهًا منفصلًا عن الدرس.

## التجارب العملية

كل تجربة يجب أن تحتوي على:

- عنوان.
- هدف.
- أدوات.
- خطوات.
- ملاحظات سلامة.
- `safetyLevel`.
- سؤال ملاحظة.
- سؤال استنتاج.

## مستويات السلامة

| القيمة             | المعنى                                    |
| ------------------ | ----------------------------------------- |
| safe_home          | آمنة للتنفيذ المنزلي بأدوات بسيطة         |
| teacher_supervised | تحتاج إشراف معلم أو راشد                  |
| lab_only           | تنفذ فقط في مختبر مجهز                    |
| not_allowed        | لا تنفذ فعليًا وتعرض كمحاكاة أو فيديو فقط |

## المحتوى المولّد بالذكاء الاصطناعي

أي محتوى يحمل `source: ai_generated` لا ينتقل إلى `status: approved` إلا بعد مراجعة بشرية صريحة.
